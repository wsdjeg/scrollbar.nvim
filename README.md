# scrollbar.vim

## Requirements

- Vim: `exists('*popup_create')`
- Neovim: `exists('*nvim_open_win')`

## Installation

Using [nvim-plug](https://github.com/wsdjeg/nvim-plug)

```lua
require('plug').add({
    { 'wsdjeg/scrollbar.vim' },
})
```
