## Install

1. Using `scrollbar.vim` in SpaceVim:

```toml
[[layers]]
  name = 'ui'
  enable_scrollbar = true
```

2. Using `scrollbar.vim` without SpaceVim:

```
Plug 'wsdjeg/scrollbar.vim'
```
